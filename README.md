# AI VIRTUAL ASSISTANT 🤖

## PROJECT OVERVIEW
This is a full stack AI Virtual Assistant web application built using HTML, CSS, JavaScript for the frontend and Python (Flask) for the backend. The project allows users to interact with a chatbot interface where messages are sent from the frontend to the backend and responses are displayed in real time.

This project helps in understanding how frontend and backend communication works in web development.

---

## FEATURES
- Chatbot style user interface
- User and bot message display
- Real-time communication between frontend and backend
- Clean and responsive UI design
- Background styling for better UI experience
- Easy to extend with AI APIs

## HOW TO RUN

1. Clone the repository:
git clone https://github.com/your-username/AI-Virtual-Assistant.git

2. Install dependencies:
pip install flask flask-cors

3. Run backend:
python app.py

4. Open frontend:
Open index.html in browser

---

## WORKING
User enters a message → JavaScript sends it to Flask backend → Backend processes it → Response is returned → Displayed in chat UI
